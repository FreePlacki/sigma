## Syntax

### Variable declaration
```
m = 5.1    kg
abcd_123a = -1
```

Units at the end, separated by one tab.

---

### Evaluating expressions
```
2m/(5pi*e)^2
```

"`*`" operator is optional, but consider the following:

```
ab = 5
a = 1
b = 2

ab  # >> 5
a*b # >> 2
```

---
### Operations
`+` addition

`-` substraction

`*` multiplication

`/` division

`^` power (use fractions for roots)

`!` factorial
