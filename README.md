## Syntax

### Variable declaration
```
abcd_123a = -1
```
Units after ":"
```
m: kg = 5.2
```

---

### Evaluating expressions
```
2m/(5pi*e)^2
```

"`*`" operator is optional, but consider the following:

```
ab = 5
a = 1
b = 2

ab  # >> 5
a*b # >> 2
```

---
### Operations
`+` addition

`-` subtraction

`*` multiplication

`/` division

`^` power (use fractions for roots)

`!` factorial
